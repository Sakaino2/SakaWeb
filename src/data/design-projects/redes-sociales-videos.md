## Resumen del Proyecto

Trabajos de edición de videos para redes sociales para restaurantes e inmobiliarias.

### Características

- Uso de motion graphics para generar dinamismo
- Mantener identidad visual para cada marca

### Proceso de Diseño

Se siguió el brief especificado por otras áreas para editar los videos y se propusieron ideas para mejorar la presentación.

### Aprendizajes

Al manejar diversos proyectos, fue importante priorizar el orden y crear un pipeline para agilizar la edición de los videos.
